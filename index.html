Política de Privacidad

Bamos al Fut

Última actualización: Febrero 2026

1. Información general

Esta Política de Privacidad describe cómo Bemuss recopila, utiliza y protege la información personal de los usuarios de la aplicación móvil Bamos al Fut, disponible para dispositivos Android.

2. Responsable del tratamiento de datos

  • Marca / Desarrollador: Bemuss
  • País de operación: Costa Rica
  • Correo de contacto: macchie-23@gmail.com

3. Información que recopilamos

La aplicación puede recopilar la siguiente información personal:

  • Nombre
  • Correo electrónico
  • Foto de perfil

4. Autenticación

La aplicación utiliza autenticación mediante correo electrónico y contraseña para la gestión de cuentas de usuario.

5. Uso de la información

  • Crear y administrar cuentas de usuario
  • Autenticación de usuarios
  • Funcionamiento de las funcionalidades principales
  • Comunicación con el usuario por correo electrónico
  • Análisis y mejora de la aplicación

6. Almacenamiento y seguridad

La información se almacena en un backend propio gestionado por Bemuss. Se aplican medidas de seguridad razonables para proteger los datos personales.

7. Compartición de datos

La aplicación no vende, alquila ni comparte información personal con terceros, salvo obligación legal.

8. Eliminación de datos

El usuario puede solicitar la eliminación de su cuenta y datos personales enviando un correo a macchie-23@gmail.com.

9. Privacidad de menores

La aplicación no está dirigida a menores de edad ni a menores de 13 años.

10. Cambios en esta política

Bemuss se reserva el derecho de modificar esta política cuando sea necesario.

11. Aceptación

Al utilizar la aplicación Bamos al Fut, el usuario acepta esta Política de Privacidad.
